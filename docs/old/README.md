拓扑，也不能直接复制命令执行；当前部署请使用上级目录的
[`deployment.md`](../deployment.md)。

- `docker-builder-*.md`：Docker 接力构建的临时救援记录。
- `done-todo-impermanence.md`：已完成的 impermanence 迁移清单。
- `ml-builder-*.md`：ml-builder 历次重装与运行日志。
- `old-fork-change-table.md`：早期 fork 差异追踪。
- `old-nixos-rollback-rescue.md`：历史回滚救援过程。
- `self-hosted-builder.md`：已替代的 NAS/Docker 构建路线。
- `upstream-replication-checklist.md`：早期原版复刻检查表。
